# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

Project participants are expected to be respectful, constructive, and focused on
the work. Harassment, personal attacks, or exclusionary behavior are not
acceptable.

Instances of unacceptable behavior can be reported through the repository's
maintainer contact path. Maintainers are expected to review reports fairly and
take proportionate action.
